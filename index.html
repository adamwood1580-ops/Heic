Troubleshooting “format not supported”
Some HEIC files can’t be decoded in-browser due to codec support limits (often HEVC). If a file fails:
  • Try a different browser/device (Safari on iPhone sometimes works via native decoding).
  • If you need 100% reliability for all HEIC variants, you’ll need a server-side converter (e.g., ffmpeg/libheif built with HEVC support).
  • For future iPhone photos: Camera settings → Formats → “Most Compatible” (captures JPEG/HEVC video changes).
